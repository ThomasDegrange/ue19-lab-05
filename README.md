### **Random jokes generator**

This is a very simple Python script that queries an API to get a random joke that will be displayed in the console.

I chose Python because it's a language whose basics I understand

• How to Install and Run the Project:
- Install Python 3.7 or later
- Install the requests module
  _pip install requests_

• How to Use the Project:
- Clone the project and then simply run the script
  _python get_a_joke.py_
- Example output:
  _Blague :  What did the beaver say to the tree?
  Réponse :  It's been nice gnawing you._
